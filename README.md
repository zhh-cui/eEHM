eEHM
====

enhanced exception handling method

This library enables treating SIGFPE, SIGSEGV, SIGINT, SIGTERM, SIGABRT, SIGILL signals as standard exception, which will help programmer catches run-time errors, especially SIGFPE and SIGSEGV signals.

You can get help [here][1].

[1]: http://zhh-cui.github.io/ "增强的异常处理方法（enhancedExceptionHandlingMethod）"
